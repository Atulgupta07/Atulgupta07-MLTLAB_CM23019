# 🤖 TensorFlow.js Model Deployment

## 📌 Objective
This project demonstrates deploying a machine learning model directly in the browser using TensorFlow.js.

## 🚀 Features
- Train a model in browser
- Save model to LocalStorage
- Reload and verify predictions
- Export model to files
- Import model and test predictions

## 🧠 Model
Simple Linear Model:
y = 3x + 1

## 🛠️ Tech Used
- TensorFlow.js
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Open index.html in browser
2. Click "Train & Save"
3. Click "Load & Verify"
4. Export and Import model

## 📊 Output
Predictions remain consistent across all steps.
