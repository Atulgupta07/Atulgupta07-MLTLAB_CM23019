How to Use

🚀 Getting Started

  • Open Task 1 — upload 5+ images per fruit category (Apple, Banana, Orange)
  • Click Extract Features — MobileNet processes each image into a 1280-dim vector
  • Click Train Classifier — custom dense layers train on top of features
  • Open Task 2 — upload a validation set, run evaluation, view confusion matrix
  • Open Task 3 — add a 4th fruit (Mango), observe accuracy shift
